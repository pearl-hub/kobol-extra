
XML/URL - Encode for XML and URL
======================


XML Encode
----------

`" -> &quot;`

`& -> &amp;`

`&apos; -> &apos;`

`< -> &lt;`

`> -> &amp;gt;`


More info: http://en.wikipedia.org/wiki/List_of_XML_and_HTML_character_entity_references
Use CDATA construct to avoid code all special chars.


URL Encode
----------

`Tab -> %09`

`Space -> %20`

`" -> %22`

`- -> %23`

`% -> %25`

`&amp; -> %26`

`( -> %28`

`) -> %29`

`+ -> %2B`

`, -> %2C`

`. -> %2E`

`/ -> %2F`

`: -> %3A`

`; -> %3B`

`< -> %3C`

`= -> %3D`

`> -> %3E`

`? -> %3F`

`@ -> %40`

`[ -> %5B`

`\ -> %5C`

`] -> %5D`

`^ -> %5E`

`' -> %60`

`{ -> %7B`

`| -> %7C`

`} -> %7D`

`~ -> %7E`
